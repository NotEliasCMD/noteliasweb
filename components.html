      11ASCII animation library
      anim/ascii.js (engine) + one plugin per anim/<name>.js
    

    
Every animation registered with ASCII.register(), mounted live via ASCII.mount() — pure text into a <pre>, no WebGL. The ones tagged · plane below are what the project “planes” mount today (see data-anim in index.html): the Work planes use neuralnet/candlestick/barchart/attention/topology. (The mirrored Personal-works planes render SVG data-story charts, not ASCII — they mount none of these.) The rest are library spares (including motherboard, cube, and the legacy dna) ready to drop into a plane or the closer. This grid builds itself from the registry, so a new anim/<name>.js (plus its <script> tag) shows up here automatically.
